# IPL Analytics Pro 🏏

An end-to-end Machine Learning web application that predicts:

* 📈 First Innings Projected Score
* 🏆 Second Innings Win Probability

## Live Demo

https://ipl-analytics-pro.onrender.com

## GitHub Repository

## Features

* Projected Score Prediction
* Win Probability Prediction
* Interactive Web Interface
* Machine Learning Models
* Real-time Predictions
* Responsive UI

* Python
* Flask

* HTML
* CSS
* JavaScript
* Git & GitHub

## Machine Learning

### Projected Score Prediction

* Algorithm: Random Forest Regressor
* Predicts the expected first innings score using live match statistics.

### Win Probability Prediction

* Algorithm: Random Forest Classifier
* Predicts the winning probability during the second innings.
