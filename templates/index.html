About IPL Analytics Pro

IPL Analytics Pro is an AI-powered cricket analytics platform developed using Machine Learning and Flask. The application predicts first innings scores and second innings win probabilities using historical IPL match data.

Projected Score Model

Random Forest Regressor predicts the final innings score.

Win Probability

Logistic Regression predicts the probability of victory.
